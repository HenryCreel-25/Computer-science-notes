## 2.循环
### 1)while循环
> while 条件表达式：
>
> 循环语句
>
> ...
>
> else:(可选)：
>
> 条件为False，循环正常结束时进行
>

```python
i = 0
while  i < 10:
    print(i)
    i+=1
else:
    print("循环结束")
```

### 2)for循环
> for 元素 in 待处理数据集:
>
> 循环体代码
>
> else:
>
> 循环结束时执行
>

```python
num = [2,3,4,5,1,9]
for i in num:
    print(i)
else:
    print("循环结束")
```



while：用于某个条件满足时一直循环，循环的次数通常未知，只关注循环开始/结束的条件

for：用于对一个已知的数据集进行遍历或已知次数的循环，关注的是遍历每一个元素

### 2）切片
切片指对操作的数据截取其中一部分的操作。列表/元组/字符串都支持切片操作

语法：序列名称[开始索引：结束索引：步长]

```python
s = [25,"zyujian",19,"xixi"]
print(s[1:3:1])
```

> 注意：
>
> + 开始索引未指定默认为0，结束索引默认为列表长度，步长默认为1
> + 不包含结束索引位置对应的元素
> + 索引采用正向/反向索引都可以
>

### 3）常用方法
| 函数 | 作用 | 例子 |
| --- | --- | --- |
| append(x) | 列表尾部追加元素x | s.append(2519) |
| insert(position,x) | 指定索引**前**插入该元素x | s.insert(0,92) |
| remove(x) | 移除列表中第一个匹配到的值 | s.remove(0) |
| pop(x) | 删除列表中指定位置的元素（索引未指定则删最后一个） | s.pop(2)/s.pop() |
| sort() | 对列表进行排序（要求元素数据类型一致） | s.sort() |
| reverse() | 反转列表元素 | s.reverse() |


```python
list1 = [1,3,5,7]
list2 = [2,4,6,8]

#1.使用+运算符
ret_list = list1 + list2
print(ret_list)

#2.使用*进行解包操作
ret_list = [*list1,*list2]
print(ret_list)
```



```python
list1 = [1,3,4,5,7,1,4,3]
ret_list = []
for num in list1:
    if num not in ret_list:
        ret_list.append(num)
print(ret_list)
```



列表推导式：按照一定规则快速生成一个列表的方法

```python
#列表名称 = [要插入列表的数据 for i in 列表]
list1 = [1,3,4,5,7,1,4,3]
s = [i for i in list1]
print(s)

#列表名称 = [要插入列表的数据 for i in 列表 if 条件]
s = [i for i in list1 if i % 2 != 0]
print(s)
```
